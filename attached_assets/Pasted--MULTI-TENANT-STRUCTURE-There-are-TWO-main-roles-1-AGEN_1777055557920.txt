
# 👥 MULTI-TENANT STRUCTURE

There are TWO main roles:

## 1. AGENCY (ADMIN)
- Can see ALL clients
- Can add new clients via API Key
- Can see aggregated metrics across all clients
- Can access any client dashboard

## 2. CLIENT (LOJISTA)
- Can only see their own data
- Has access to full dashboard (filtered)

---

# 🔐 AUTH SYSTEM

- Login screen
- Role-based access:
  - ADMIN
  - CLIENT

---

# 🏢 AGENCY PANEL (ADMIN VIEW)

## Screen: Client Management

### Features:
- List of clients
- Button: "Add New Client"

### Add Client Flow:
- Input fields:
  - Client Name
  - API Key
- On submit:
  - System fetches data automatically from API
  - Creates new client environment

### Client List Table:
- Client Name
- Total Revenue
- Orders
- ROAS
- Leads
- Approval Rate

Clicking a client → opens full dashboard scoped to that client

---

## Screen: Global Overview (Agency Dashboard)

### KPIs:
- Total Revenue (All clients)
- Total Orders
- Total Ad Spend
- Global ROAS
- Total Leads
- Approved Leads

### Charts:
- Revenue over time (all clients)
- Leads over time

### Insights:
- Best performing client
- Worst performing client
- Clients with drop in conversion
- Clients with growth
